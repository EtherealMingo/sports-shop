## 文档使用指南

1. **开发前**: 先读 [01-项目概述.md](01-项目概述.md) 了解全局，再读对应模块文档
2. **数据库**: 建表时参考 [05-数据库设计.md](05-数据库设计.md) 获取完整表结构
3. **API**: 接口定义在各模块文档的"API 接口"章节
4. **UI**: 设计页面时参考 [06-UI设计规范.md](06-UI设计规范.md)
5. **Token 节省**: 按需读取对应模块文档，不要一次性加载全量需求
