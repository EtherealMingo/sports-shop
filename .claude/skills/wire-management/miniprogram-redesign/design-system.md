## 一、设计系统

### 1.1 色彩系统

```css
/* 主色调 - 运动活力橙 */
--primary: #FF6B35;           /* 活力橙 - 主按钮、重点标识 */
--primary-light: #FF8C61;     /* 浅橙 - 悬停状态 */
--primary-dark: #E55A2B;      /* 深橙 - 按下状态 */

/* 辅助色 */
--secondary: #00D9FF;         /* 电光蓝 - 次要按钮、链接 */
--accent: #FFD700;            /* 金牌黄 - 积分、等级、成就 */
--success: #00E676;           /* 活力绿 - 成功状态 */
--warning: #FFC107;           /* 警告黄 */
--error: #FF5252;             /* 热情红 - 错误状态 */

/* 背景色 */
--bg-primary: #0A0E27;        /* 深空蓝黑 - 主背景（运动科技感） */
--bg-secondary: #12183A;      /* 深蓝 - 卡片背景 */
--bg-tertiary: #1A237E;       /* 靛蓝 - 次级背景 */
--bg-card: rgba(255,255,255,0.1); /* 玻璃卡片背景 */

/* 文字色 */
--text-primary: #FFFFFF;      /* 纯白 - 主文字 */
--text-secondary: #B0BEC5;    /* 浅灰蓝 - 次要文字 */
--text-muted: #78909C;        /* 中灰 - 辅助文字 */

/* 渐变 */
--gradient-primary: linear-gradient(135deg, #FF6B35 0%, #FF8C61 100%);
--gradient-card: linear-gradient(135deg, #FF6B35 0%, #E55A2B 100%);
```

### 1.2 字体规范

```css
/* 标题 */
--font-title: 36-48rpx, PingFang SC Bold;

/* 正文 */
--font-body: 28-32rpx, PingFang SC Regular;

/* 数字（积分、金额） */
--font-number: DIN Alternate Bold / Oswald, 运动感;
```

### 1.3 间距规范

```css
/* 页面边距 */
--page-padding: 24rpx;

/* 卡片间距 */
--card-gap: 16rpx;

/* 元素间距 */
--element-gap: 12rpx;
```

### 1.4 圆角规范

```css
/* 大圆角 - 卡片、弹窗 */
--radius-lg: 24rpx;

/* 中圆角 - 按钮 */
--radius-md: 16rpx;

/* 小圆角 - 标签、输入框 */
--radius-sm: 8rpx;
```

## 七、设计原则

1. **功能前置**：核心功能（积分、预约、扫码）在首页显要位置展示
2. **商家入口明显**：在「我的」页面顶部设置醒目的商家端入口
3. **身份区分明确**：用户端和商家端使用不同的配色和布局
4. **信息层次清晰**：使用卡片、间距、颜色区分信息层级
5. **操作路径短**：核心功能最多 2 次点击可达
